## 三、部署步骤（30 分钟）

### Step 1：买服务器

1. 打开 [腾讯云轻量应用服务器](https://cloud.tencent.com/product/lighthouse)
2. 选 **2核4G / 系统镜像 Ubuntu 22.04** → ¥68/月
3. 地域选 **广州**（离用户近）
4. 购买后记录 **公网 IP**

### Step 2：买域名 + 备案

1. 腾讯云域名注册 → `yatitiyu.com`（或你想要的）
2. **备案**（国内服务器必须备案，约 15 个工作日）
3. DNS 解析：添加 A 记录 → 指向服务器 IP

> ⚠️ 备案期间可用 `http://IP:8000` 临时测试

### Step 3：登录服务器

```bash
# Mac 本地终端
ssh root@你的服务器IP

# 装 Docker（腾讯云已预装可跳过）
curl -fsSL https://get.docker.com | bash
```

### Step 4：拉代码 + 启动

```bash
# 服务器上
git clone https://github.com/YOU        /yati-edu-project.git
cd yati-edu-project

# 配置环境变量
cp backend/.env.example backend/.env
vim backend/.env   # 填 SECRET_KEY 和 One API Token

# 一键启动
docker compose up -d

# 验证
curl http://localhost:8000/health
# → {"status":"ok"}
```

### Step 5：配置 SSL（Let's Encrypt 免费）

```bash
# 安装 certbot
apt install certbot -y

# 申请证书
certbot certonly --webroot -w /var/www/certbot \
  -d api.yatitiyu.com

# 证书自动复制到 deploy/ssl/
cp /etc/letsencrypt/live/api.yatitiyu.com/fullchain.pem deploy/ssl/
cp /etc/letsencrypt/live/api.yatitiyu.com/privkey.pem deploy/ssl/

# 重启 nginx
docker compose restart nginx
```

### Step 6：验证

```bash
# HTTPS 访问
curl https://api.yatitiyu.com/health
# → {"status":"ok"}

# 打开 Swagger 文档
open https://api.yatitiyu.com/docs
```

## 五、安全 Checklist

- [ ] 修改 SSH 默认端口（22 → 2222）
- [ ] 配置防火墙（仅开放 80/443/2222）
- [ ] `.env` 文件权限设为 600
- [ ] 数据库定期备份（cron + COS）
- [ ] 开启腾讯云安全组
- [ ] Let's Encrypt 证书自动续期（cron）

```bash
# 自动续期 cron
0 3 * * * certbot renew --quiet && cp /etc/letsencrypt/live/api.yatitiyu.com/*.pem /path/to/deploy/ssl/ && docker compose restart nginx
```
